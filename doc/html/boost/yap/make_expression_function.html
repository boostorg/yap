Function template make_expression_function

boost::yap::make_expression_function

Synopsis

// In header: <boost/yap/expression.hpp>


template<typename Expr> auto make_expression_function(Expr && expr);

Description

Returns a callable object that expr has been moved into. This is useful for using expressions as function objects.

Note

make_expression_function() is only valid if Expr is an expression.
